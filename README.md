band-name
=========

World's finest band name generator


### Usage
```
> const generate = require('band-name').generate;

> generate(); //
'The Ability Labor'

> generate('metal');
'Impaled Blood Gargoyle'

> Array(10).fill().forEach(() => console.log(generate('indie')))
The Curiosity Topic
Failuremancy
Leslie's Leopard
Stupiditymeister

Harmony's Curiosity
Strictness Province

```
